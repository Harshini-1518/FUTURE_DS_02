# FUTURE_DS_02 – Customer Retention & Churn Analysis

# Project Overview :
This project focuses on analyzing customer retention and churn patterns for a subscription-based business. The goal is to identify the reasons behind customer churn, evaluate retention trends, and provide actionable recommendations to improve customer loyalty and reduce customer loss.

# Objectives :
* Analyze customer retention and churn rates.
* Identify key factors contributing to customer churn.
* Evaluate customer lifetime value and subscription performance.
* Generate insights to support business decision-making.
* Recommend strategies to improve customer retention.

# Tools Used :
* Microsoft Excel
* Pivot Tables
* Charts & Visualizations
* KPI Analysis

# Analysis Performed :
* Active vs Churned Customer Analysis
* Churn Reason Analysis
* Subscription Plan Performance Analysis
* Regional Customer Analysis
* Customer Spending Analysis
* Retention Rate Calculation
* Dashboard Creation

# Key Metrics :
* Total Customers: 10
* Active Customers: 6
* Churned Customers: 4
* Retention Rate: 60%
* Churn Rate: 40%

# Key Insights :
* Price sensitivity was one of the major reasons for customer churn.
* Poor service quality contributed significantly to customer loss.
* Premium plan customers generated higher revenue compared to other plans.
* Customer retention varied across different regions.
* Long-term customers showed higher spending patterns.

# Recommendations :
1. Improve customer support and service quality.
2. Offer loyalty rewards and retention incentives.
3. Review pricing strategies to remain competitive.
4. Focus on retaining high-value customer segments.
5. Monitor churn trends regularly using dashboards and KPIs.
